# Surface Design

## Scope

This skill defines the design language of Surface.

It provides the principles, visual grammar and component specifications required to build a consistent institutional interface.

## References

Every Surface UI task MUST load `references/design-principles.md` and `references/visual-language.md`. Additionally, load `references/layout.md` and `references/typography.md` for rendered screens or components; `references/color.md` for semantic state or interactive controls; `references/motion.md` for animations or transitions; `references/navigation.md` for routes, links, or search; and `references/identity.md` whenever an Abstract Account or derived identifier is shown.
