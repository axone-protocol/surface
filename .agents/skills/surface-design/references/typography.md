# Typography

Typography communicates institutional roles.

Different typefaces distinguish different kinds of information.

Typography must never be decorative.

## Principles

Every typeface has a semantic role.

Do not introduce additional typefaces without assigning them a clear institutional purpose.

## Institutional text

Use the institutional typeface for:

- assertions;
- titles;
- institutional language.

Institutional text is intended to be read.

## Technical text

Use the monospace typeface for:

- references;
- addresses;
- hashes;
- timestamps;
- identifiers;
- protocol values.

Technical text is intended to be inspected.

## Hierarchy

Hierarchy follows institutional importance.

Institutional meaning takes precedence over technical detail.

## Consistency

The same information must always use the same typographic treatment.

Equivalent concepts must remain visually equivalent.

## Density

Prefer readable typography over compact typography.

Reduce information before reducing legibility.

## Emphasis

Use typography to establish hierarchy.

Avoid emphasis through excessive weight, size or color.

## Alignment

Typography follows layout semantics.

Text should align according to its institutional role, not visual convenience.

## Wrapping

Institutional assertions should read naturally.

Technical identifiers should remain compact.

Prefer compact representations over wrapped identifiers.
